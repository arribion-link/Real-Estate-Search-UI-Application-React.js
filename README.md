## Real Estate Home search Web application

### Getting 

Get the project
```bash
git clone https://github.com/arribion-link/Real-Estate-Search-UI-Application-React.js.git
```

Download the dependancies
```bash
npm install
```

Running the project
```bash
npm run dev
```
